# PromptPro AI — Multilingual Prompt Builder

PromptPro turns rough ideas into clear, structured, copy-ready prompts. It supports multilingual input and produces bilingual results without requiring a login, API key, paid credits, or software installation.

## Live project and demo

- [Try PromptPro](https://73629.genesis.estage.com/)
- [Watch the 48-second demo](https://youtu.be/6Z8W8HsmstE)

## What PromptPro does

- Guides users from a rough idea to a stronger AI prompt
- Supports English and Spanish workflows
- Produces bilingual, copy-ready output
- Offers tone, detail, and formatting choices
- Includes token-saving prompt controls
- Runs directly in the browser

## Built with

- React
- Vite
- JavaScript
- Tailwind CSS
- Motion

This repository contains the production build submitted to OpenAI Build Week. The application is client-side and requires no environment variables, credentials, database, or external API key.

## How Codex and GPT-5.6 were used

Codex and GPT-5.6 supported the build as an engineering and product-development partner. Their work included:

- Reviewing the application structure and helping prioritize the submission-ready MVP
- Diagnosing interaction problems, including overlapping controls that intercepted clicks
- Refining responsive layout, spacing, readability, and footer presentation
- Improving multilingual user flow and clarifying interface copy
- Preserving working features while narrowing risky last-minute changes
- Testing the published experience and preparing the submission documentation

The product direction, feature decisions, branding, testing, and final approval remained human-directed.

## Run locally

From the repository folder, run:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080` in a browser.

## Suggested judging flow

1. Open the live project.
2. Review the English interface.
3. Change the language to Spanish.
4. Enter a rough idea and select the desired prompt options.
5. Generate the prompt and review the bilingual result.

No login, credentials, API key, or paid credits are required.
