
# PromptPro AI — Multilingual Prompt Builder

PromptPro turns rough ideas into clear, structured, copy-ready AI prompts. It supports multilingual input and bilingual results without requiring a login, API key, paid credits, or installation.

## Live project and demo

- Live app: https://73629.genesis.estage.com/
- Video demo: https://youtu.be/6Z8W8HsmstE

## Key features

- English and Spanish workflows
- Bilingual, copy-ready prompt results
- Tone, detail, and formatting controls
- Token-saving prompt options
- No login, API key, or paid credits required

## Built with

React, Vite, JavaScript, Tailwind CSS, and Motion.

## How Codex and GPT-5.6 were used

Codex and GPT-5.6 supported PromptPro as engineering and product-development partners. They helped:

- Review the application and prioritize the submission-ready MVP
- Diagnose interaction problems, including overlapping controls that intercepted clicks
- Improve responsive layout, spacing, readability, and footer presentation
- Refine the multilingual workflow and interface copy
- Preserve working features while limiting risky last-minute changes
- Test the published experience and prepare submission documentation

Product direction, branding, feature decisions, testing, and final approval remained human-directed.

## Testing instructions

1. Open the live app.
2. Review the English interface.
3. change the language to Spanish.
4. Enter an idea and select the desired options.
5. Generate and review the bilingual prompt.
